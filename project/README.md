# AI Agent Studay Group Project Overview

## Objectives
This exercise aims to create expertise, tool sets for building AI agents in on-premise and air-gapped environment.

## Environment
There are two environment with a GPU. Both provides Jupyter Notebook interface.
* A Kubernetes cluster with 1x GPU
* Dell Max Pro with GB 10, Docker system

## Projects 
* AI agent for cluster / application analysis
* API server to provide data extraction from publicly available data such as corporate filings
